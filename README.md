Portfolio
===

Portfolio achieve in ReactJs editable through a json file

![Home](http://garreca.fr/img/home.png)

Install
-------

In the project's directory, run the following commands:

    $ gulp

Releasing to Production

HTML ,  SCRIPT and css compressed.

In the project's directory, run the following commands:

    $ gulp prod
